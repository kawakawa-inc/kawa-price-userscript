# KAWA Price Userscript

![](https://github.com/kawakawa-inc/kawa-price-userscript/blob/main/usage.gif?raw=true)

This Userscript for Prosperous Universe adds a button to load KAWA prices into contract drafts.

## Installation

1. Install the [Tampermonkey](https://www.tampermonkey.net/) browser extension in your web browser
2. Click [this link](https://github.com/kawakawa-inc/kawa-price-userscript/raw/refs/heads/main/kawa-prices.user.js) to install or update the userscript.
   Alternatively, you can also click on kawa-prices.user.js and then click the "Raw" button
